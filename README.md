## ğŸš€ Installation

### Prerequisites

- A web browser (Chrome, Firefox, Safari, Edge, etc.)
- Git (for cloning the repository)
- A code editor (VS Code, Sublime Text, etc.) - optional

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/soumitra69/collage_website.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd collage_website
   ```

3. **Open in browser**
   - Simply open `index.html` in your preferred web browser
   - Or use a local development server:
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Python 2
     python -m SimpleHTTPServer 8000
     
     # Using Node.js (if you have http-server installed)
     npx http-server
     ```

4. **View the website**
   - If using a local server, navigate to `http://localhost:8000` in your browser

## ğŸ’» Usage

### For Viewing
Simply visit the live website at [https://soumitra69.github.io/collage_website/](https://soumitra69.github.io/collage_website/)

### For Development
1. Clone the repository as described in the installation section
2. Make your changes to the HTML, CSS, or JavaScript files
3. Test locally by opening `index.html` in your browser

### Customization

To customize this website for your own college:

1. **Update Content**: Modify the text in `index.html`
2. **Change Images**: Replace images in the `Assets/` folder
3. **Modify Styles**: Edit `css/style.css` to match your branding
4. **Update Testimonials**: Change the testimonial content and images
5. **Add/Remove Sections**: Modify `index.html` structure as needed
